Minha primeira página usando o JavaScript

Variável inteira

Esse tipo de variável armazena valores inteiros, que são os número que não possuem uma vírgula

Variável texto

Esse tipo de variável armazena valores escritos entre parênteses, que isso a define ser um tipo de variável do tipo string, todo o conteúdo dentro desses parênteses vai aparecer como um texto para o usuário, mesmo sendo números

Variável lógica

Esse tipo de variável aceita valores lógicos, "true" ou "false", que são os valores conhecidos pelo computador sendo "0" e "1".

Variável sem valor definido

Este apenas se declara a variável, não atribuindo-a nenhum valor.
